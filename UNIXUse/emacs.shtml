Using Emacs

  • Getting started
    • Get help C-h (Hold down CTRL and press h)
    • Emacs manual C-h r
    • Emacs tutorial C-h t
  • Dealing with files
    • open a file C-x C-f
    • save a file under a given name C-x C-w
    • save a give file C-x C-s
  • Editing
    • undo function C-_
    • search forward C-s
    • search backward C-r
    • mark region C-space
    • kill a region C-k
    • kill a region C-w
    • replace text M-% (`M-' means use the Meta (or Alt) key. Alternatively press esc)
  • Using windows
    • close window C-x 0
    • close all windows except active one C-x 1
    • split the active window vertically C-x 2
    • split the active window horizontally C-x 3
    • change active window to next window C-x o
  • Leaving emacs
      terminating emacs C-x C-c
  • For lazy ones
    • Activate menubar M-`
